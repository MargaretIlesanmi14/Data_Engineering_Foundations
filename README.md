# Data_Engineering_Foundations
This repository contains my hands-on learning journey into Data Engineering using Python. It captures foundational concepts, practical code examples, and simple data manipulation techniques that form the building blocks of data engineering.

# Overview
This project is a Jupyter Notebook (Data_Engineering_Learnings_1.ipynb) that walks through:

Python fundamentals
Control flow (conditionals & loops)
Functions
File handling
Introduction to data manipulation with pandas

# Concepts Covered
1. Variables & Data Types
Strings, integers, booleans

2. Conditionals
if, elif, else

3. Loops
for loops (iteration)
while loops (condition-based execution)

4. SQLite and SQL for data querying

5. Functions
Creating reusable functions
Examples:
Greeting function
Addition function

6. File Handling
Writing to files (.txt)
Reading from files

7. Python Libraries
Introduction to:
pandas
numpy 

8. DataFrames with pandas
Creating DataFrames
Working with structured data

9. Data Ingestion: Web Scraping, Beautiful Soup

10. Data Pipelines - Writing a Reusable Python ETL Script, Logging and Error Handling

# Key Learnings:
- Understanding core Python concepts is essential for data engineering
- File handling is foundational for data ingestion
- pandas is a powerful tool for working with structured data
- Writing clean, reusable functions improves code quality
